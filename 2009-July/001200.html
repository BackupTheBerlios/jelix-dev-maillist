Dear commiters,

the subversion repository is now closed for ever. You can still do a 
checkout ( http://svn.jelix.org ), but you cannot commit on it.

Now, the official repository is a Mercurial repository at 
http://hg.jelix.org .

I didn't finish the access configuration so  you cannot push your 
commits on it, but you can pull from it. Try Mercurial! 
http://hg-scm.org/  or http://mercurial.selenic.com

example to get the source:

hg clone http://hg.jelix.org/jelix-trunk


For windows users, you have tortoiseHg: 
http://bitbucket.org/tortoisehg/stable/wiki/Home


Laurent
