Hello,

I decided that we should release Jelix 1.1 as soon as possible, since 
the trunk is stable enough.

So I canceled the 1.1 beta 2 milestone, and the next release will be a 
release candidate of jelix 1.1.

Only features and improvements described in tickets currently targeted 
to Jelix 1.1 are accepted. No more other new features will be included 
in the trunk until Jelix 1.1.

Please help to close all the remaining tickets.

Laurent
